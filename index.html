Estrutura de Página

  • HTML
  • HEAD
  • BODY

Aprendi sobre a estrutura fundamental de uma página HTML, que é composta por várias tags-chave. Aqui está uma breve explicação do papel de cada uma dessas tags:

HTML: A tag html envolve todo o conteúdo da página e marca o início do documento HTML. É o elemento raiz que contém todos os outros elementos HTML.

HEAD: A tag head é usada para conter metadados e informações sobre o documento, como links para folhas de estilo CSS, scripts JavaScript e configurações essenciais, mas não visíveis na página.

Vale mencionar que a tag só aparece uma vez no início do documento HTML e não deve ser repetida na página. É exclusivamente usada para informações sobre o documento, não para o conteúdo visível.

BODY: A tag body delimita o início do conteúdo visível da página. Ela envolve todos os elementos que os visitantes veem quando acessam a página, como texto, imagens, links e outros elementos.

Marcação de Conteúdo

A marcação de conteúdo HTML é o processo de adicionar estrutura e significado ao conteúdo de uma página da web por meio do uso de tags HTML. Essas tags são elementos especiais que informam ao navegador como o conteúdo deve ser interpretado e exibido. A marcação de conteúdo é fundamental para a construção de páginas web, pois não apenas organiza o conteúdo de maneira lógica, mas também influencia a acessibilidade, a formatação e a indexação pelos motores de busca.

Aqui estão algumas das principais tags de marcação de conteúdo HTML e seus usos:

P: Parágrafo, usado para criar parágrafos de texto. O texto dentro dessa tag é automaticamente formatado com espaçamento adequado entre os parágrafos.

H1 ao H6: Títulos, usados para criar cabeçalhos ou títulos de diferentes níveis de importância. H1 é o título mais importante e H6 é o menos importante.

A: Link, usado para criar links para outras páginas da web ou recursos. O atributo href especifica o URL de destino do link.

IMG: Imagem, usado para incorporar imagens na página. O atributo src especifica o caminho da imagem, e o atributo 'alt' fornece uma descrição para acessibilidade.

UL: Lista não ordenada, usado para criar listas de itens que não têm uma ordem específica. Cada item é marcado com LI.

OL: Lista ordenada, semelhante ao UL, mas cria listas com itens numerados ou com marcadores ordenados.
